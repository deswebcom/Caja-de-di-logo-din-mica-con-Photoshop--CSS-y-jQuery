Este es un ejemplo de realización de una caja de diálogo dinámica con jQuery.

Es un código de un ejemplo de un videotutorial de desarrolloweb.com, que puedes encontrar en esta dirección:

http://www.desarrolloweb.com/articulos/video-caja-dialogo-dinamica.html

Para crear la caja de diáloco con un efecto de transparencia en el borde usamos imágenes semitransparentes con PNG. Esto ya no sería lo más conveniente, puesto que en la actualidad se pueden crear efectos similares sin necesidad de usar imágenes, simplemente con CSS y transparencia por canal Alpha, o incluso con sombras creadas con boxshadow. Toda la parte de jQuery sería perfectamente válida y el conjunto del contenido de esta práctica muy útil con fines didácticos.

Puedes saber más sobre CSS en el manual:

http://www.desarrolloweb.com/manuales/css3.html
